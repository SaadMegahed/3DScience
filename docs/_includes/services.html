Frequently asked questions

Credit

20.S947 is an IAP course rather than an activity and is worth 2 credits.

Open-source

All course materials are available on the course github and all lectures will be recorded and shared on YouTube with a slightly delay.

Grading Policy

20.S947 is meant to be a fun laid-back course on all things scientific rendering.

Pre-requisites

There are no pre-requesites! We will start from the must basic concepts of 3D rendering in Blender.

Register

You can pre-register for the course here: http://student.mit.edu/cgi-docs/student.html.

Suggestions

As this is the first iteration of the course all recommendations are welcome!
